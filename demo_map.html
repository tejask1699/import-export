🔧 To Enable Real Google Maps:

  1. Get a Google Maps API key from Google Cloud Console
  2. Enable "Maps JavaScript API" in your project
  3. Replace YOUR_GOOGLE_MAPS_API_KEY in tracking.html line 511
  4. Restart your Flask application
  5. Visit /tracking to see the real Google Maps

Note: Google Maps has a free tier of 28,500 map loads per month.
